Samples folder
--------------

Coloca aquí archivos de vídeo de ejemplo (.mp4, .mkv, .avi, .mov, .jpg/jpeg) para usar con la aplicación PublisherUI.

Uso:
- Ejecuta VideoGateway.PublisherUI, pulsa "Examinar..." y selecciona esta carpeta.
- Selecciona un fichero y pulsa "Iniciar publicación" para que la app lance ffmpeg y publique a la URL RTSP indicada.

Requisitos:
- ffmpeg debe estar instalado y en el PATH para que Publisher pueda publicar.
- Para visualizar el stream en el Subscriber puedes usar VLC o ffplay.
